# windows-unattended-install


Doku und zusammenstellung für eine automatische Windows installation inkl. automatischer Installation von Programmen nach erstem Logon.

Generator für die "unattended.xml"
https://schneegans.de/windows/unattend-generator/

unter"DefaultUser scripts" wird das script hinzugefügt, welches die Programme nach der ersten Anmeldung installiert.
